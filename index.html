HOW TO INSTALL YOANIME!

YoAnime are not available on the Google Play Store and thus need to be installed manually using the following steps:

1. Allow app installs from "unknown sources". To allow app installs from non-Google play apps, open Settings of your android phone >> Applications >> Unknown sources.

2. Download and install YoAnime!.apk

3. Enjoy watching HD Anime!
